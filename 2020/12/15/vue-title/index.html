修改Vue APP网页title

二羊话叹   in Lincang
Tue Dec 15 2020
  • Vue
  • title
  • htmlWebpackPlugin

关于vue的title标签中出现的htmlWebpackPlugin.options.title

# 问题

使用vue cli快速创建的项目，编译运行网页标签title显示为项目名 比如新建vue项目dyapp，运行后网页标签title显示dyapp

希望修改title，但查看源码后发现title元素为htmlWebpackPlugin.options.title，不适合直接修改

# 解决

按照vue cli的官方文档，在项目根目录创建vue.config.js文件，编写如下代码

// vue.config.js
module.exports = {
  chainWebpack: config => {
    config
      .plugin('html')
      .tap(args => {
        args[0].title= 'new title' // 你想修改的title名
        return args
      })
  }
}

webpack在打包的时候会自动扫描是否有vue.config.js这个文件，并将其中的内容与已经设置好的webpack内容合并。
